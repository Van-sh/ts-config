# TS-Config

My Config files

## Installation

```sh
pnpm add @1sh/config
```

```sh
bun add @1sh/config
```

```sh
yarn add @1sh/config
```

```sh
npm i @1sh/config
```

## Usage

### Prettier

prettierrc.config.ts

```ts
import { prettierConfig } from "@1sh/config";

export default prettierConfig({
   // custom overrides
});

```
